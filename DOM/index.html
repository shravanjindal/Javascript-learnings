HTML code is stored as Document Object Model (DOM), and JS uses this DOM to access HTML

Window Object

The window object represents an open window in a browser.

This object is brower's object and is a global object, which is accessible by javascript.

console.log, alert are properties of window object only

DOM

Browser Creates a Document object model of HTML Page

We can access it - window.document

Note : we always link script.js before ending body tag in HTML, because if we do it in head tag, body part wont be accessible in the document object.

If script.js linked in head tag : console.dir(document.body) = null

DOM Manipulation

How to access elements

  • Access using Id - document.getElementById("");
  • Access using class - document.getElementsByClassName("");
  • Access using tag - document.getElemntsByTag("")

  • Query Selector - document.querySelector(""); || document.querySelectorAll("");
    • Id => #myId
    • Class => .myClass
    • Tag => tagName

document.getElementById("") returns object
document.getElementsByClassName("") returns HTML Collection, which is like an array of objects.
document.querySelectorAll("") returns nodelist

Some Properties

  • .tagName - return the tage name
  • .innerText - returns a string of the text content of node and its children
  • Navigation
    • .firstChild - return first child of the node
    • .lastChild - return last child of the node
    • .children - return array of all children
  • .innerHTML - returns a string of HTML contents of the element
  • .textContent - return text of hidden elements also

Attributes

  • .getAttribute("attr");
  • .setAttribute("attr","value");
  • .classList - .add .remove

Style

  • node.style => returns object = css element of the node

Insert & delete Elements

First create the element => let el = document.createElement("div");

  • node.append() - adds at the end of the node
  • node.prepend() - adds at the start of the node
  • node.before() - adds before the node
  • node.after() - adds after the node

  • node.remove()
